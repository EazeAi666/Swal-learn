## 🌍 What is SWAL-Learn?

**SWAL-Learn** is the free AI education platform of the **SWAL Organization** — *Shaping Worlds through AI & Learning*. We believe that quality AI education should be accessible to everyone, everywhere, regardless of financial background.

No subscriptions. No paywalls. No gatekeeping.  
Just knowledge — for free, forever.

## 📚 Course Tracks

| Track | Level | Description |
|-------|-------|-------------|
| 🤖 **AI Basics / Intro to AI** | Beginner | Understand what AI is, how it works, and why it matters |
| ✦ **Prompt Engineering** | Intermediate | Master the art of communicating with AI models |
| ⚡ **Building AI Apps** | Intermediate | Build real AI-powered apps using the Claude API |
| 🧠 **Machine Learning** | Advanced | Foundations of ML — data, models, training & evaluation |

## 🤝 Contributing

SWAL-Learn is open source and welcomes contributions from everyone!

### How to Contribute
1. **Fork** this repository
2. **Create** a branch: `git checkout -b feature/your-lesson`
3. **Add** your lesson or improvement
4. **Commit** your changes: `git commit -m "Add lesson: [topic]"`
5. **Push** and open a **Pull Request**

### Contribution Guidelines
- Keep lessons beginner-friendly and jargon-free
- Add code examples wherever possible
- Include a summary and learning objectives per lesson
- Follow the existing folder structure
