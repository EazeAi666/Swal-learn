Module 1: APIs & AI Services

What is an API?

An API (Application Programming Interface) is how your app talks to AI services. Think of it like a waiter: you order food (send request), kitchen prepares it (AI processes), waiter brings it back (you get response).

Popular AI APIs

• OpenAI API — GPT-4, ChatGPT capabilities
• Claude API — Anthropic's AI assistant
• Google Gemini — Google's AI models
• Hugging Face — Open-source AI models

API Key Basics

Every API requires a key (like a password). Never share your API key or put it in public code!

// Example: API request structure
fetch('https://api.openai.com/v1/chat/completions', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    model: 'gpt-3.5-turbo',
    messages: [{role: 'user', content: 'Hello!'}]
  })
})
