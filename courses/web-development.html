How the Web Works
Browsers, servers, HTTP, and URLs explained
Unlocked ⌄

The Big Picture

When you type a URL and press Enter, a chain of events happens: your browser looks up the domain's IP address (DNS), connects to the server, sends an HTTP request, and the server sends back an HTML file. Your browser then renders that HTML as a visual page.

Key Concepts

  • URL — The address of a web resource (https://example.com/page)
  • DNS — Translates domain names to IP addresses (like a phone book)
  • HTTP/HTTPS — The protocol for transferring web data
  • Browser — Reads HTML/CSS/JS and renders it visually
  • Server — A computer that stores and sends web files

HTML: Structure of a Web Page
Tags, elements, and semantic HTML
Locked ⌄

What is HTML?

HTML (HyperText Markup Language) provides the structure of a web page. It uses tags — written in angle brackets — to define elements: headings, paragraphs, images, links, buttons, and more.

Basic Structure

Every HTML file has: <!DOCTYPE html>, <html>, <head> (meta info, title, links), and <body> (visible content).

Common Tags

  • <h1> to <h6> — Headings (h1 is the largest)
  • <p> — Paragraph
  • <a href='...'> — Link
  • <img src='...'> — Image
  • <div> — Container block
  • <span> — Inline container

Semantic HTML

Use meaningful tags: <header>, <nav>, <main>, <article>, <footer>. This improves accessibility and SEO.

CSS: Styling Your Website
Colors, fonts, spacing, borders, and more
Locked ⌄

What is CSS?

CSS (Cascading Style Sheets) controls how HTML elements look — their color, font, size, spacing, position, and animation. Without CSS, every website would look like a plain text document.

Ways to Add CSS

  • Inline — style='...' attribute (avoid except for quick tests)
  • Internal — <style> block in the <head>
  • External — Separate .css file linked with <link> (recommended)

Key CSS Properties

  • color — text colour
  • background-color — background
  • font-size — text size (px, rem, em)
  • margin — space outside element
  • padding — space inside element
  • border — border around element

CSS Selectors

Target elements by tag (p), class (.classname), or ID (#idname). Classes can be reused; IDs should be unique.
