🎨 Module 2: CSS Styling Mastery

Transform plain HTML into beautiful, responsive, and modern web experiences.

1. CSS Variables (Custom Properties)

Create maintainable designs with CSS variables. Change your entire color scheme in one place!

:root { --primary: #6366f1; --spacing: 1rem; } .button { background: var(--primary); padding: var(--spacing); }

2. Flexbox & Grid Layouts

Modern CSS layout systems that replace float-based hacks.

  • Flexbox: One-dimensional layouts (navigation, centering)
  • Grid: Two-dimensional complex layouts (dashboards, galleries)

3. Responsive Design with Media Queries

Make your site work on phones, tablets, and desktops seamlessly.

@media (max-width: 768px) { .container { grid-template-columns: 1fr; } }
