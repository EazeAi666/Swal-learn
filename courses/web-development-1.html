🌐 Module 1: HTML Foundations

Lesson 1: Semantic HTML

Professional web development is about Semantics. Search engines (SEO) and SWAL AI understand your code better when you use tags that have meaning.

  • <main>: Wraps the core content of your page.
  • <section>: For distinct thematic groupings.
  • <article>: For content that can stand alone.
💡 Pro Tip: Think of semantic HTML as giving your content a clear outline, like chapters in a book.

Lesson 2: Meta Tags & SEO

The <head> section controls how your site appears in search results and social media.

<meta property="og:title" content="My Awesome Site">
<meta property="og:image" content="preview.jpg">

Lesson 3: Accessibility

Great websites work for everyone. Semantic HTML is the foundation of web accessibility (a11y).

AI Insight: Most languages use {curly braces}. HTML uses semantic tags. This forces you to write clean, organized code that screen readers can navigate easily.
