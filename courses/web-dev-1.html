🏗️ Module 1: Advanced HTML Structure

Master the art of building semantic, accessible, and SEO-friendly websites with SWAL AI.

1. Beyond the <div>: Semantic HTML

Professional web development is about Semantics. Search engines (SEO) and assistive technologies understand your code better when you use tags that have meaning.

  • <main>: Wraps the core content of your page. Only one per page.
  • <section>: For distinct thematic groupings with their own heading.
  • <article>: For content that can stand alone (blog posts, news stories).
  • <aside>: For tangentially related content (sidebars, call-out boxes).

2. The Meta-Universe & Open Graph

The <head> section controls how your site appears on search engines and social media. Open Graph (OG) tags determine preview cards on Facebook, LinkedIn, and Twitter.

  • <meta name="description">: Search engine snippet (150-160 chars)
  • <meta property="og:title">: Social media title
  • <meta property="og:image">: Social preview image

3. Accessibility First

Great websites work for everyone. Semantic HTML is the foundation of web accessibility (a11y).

  • Use <button> for clickable actions (not styled divs)
  • Always include alt text for images
  • Ensure proper heading hierarchy (h1 → h2 → h3)
