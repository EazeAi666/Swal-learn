SYSTEM: BUG_REPORTING

11 // The Anatomy of a Bug Report

When reporting a technical error, you must be precise. Avoid saying "It's broken." Instead, use the Expected vs. Actual result format.

[BUG] Login Page Timeout

Steps to Reproduce: Click 'Login' on mobile.
Expected Result: User dashboard opens.
Actual Result: Screen stays white for 10 seconds.
SYSTEM: HOW_TO_GUIDES

12-15 // Writing Steps & Procedures

Mastering the "Imperative" mood (Command words). Professional documentation uses direct verbs.

  • Good: "Open the terminal and type the command."
  • Bad: "You should maybe try to open the terminal if you can."
SYSTEM: GLOSSARY_BUILDING

16-19 // Common Tech Slang & Idioms

Learning phrases used in Silicon Valley and London tech hubs:

  • "On the same page": To have the same understanding.
  • "Bandwidth": Having the time/mental space to do a task.
  • "Deep dive": A thorough investigation into a problem.
